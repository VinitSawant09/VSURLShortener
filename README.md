# VSURLShortener is a pet project which provides a URL shortening technique similar to tinyurl and bit.ly

Technologies used:
1) Java 1.8
2) Spring 5.1
3) Hibernate ORM 5.4.21
4) HSQL DB 2.4.0 (In memory database)
5) JUnit 5
